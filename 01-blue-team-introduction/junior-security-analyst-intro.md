## Objective
Understand the role of a Junior Security Analyst and how a Security Operations Center (SOC) works.

---

## Key Concepts

### What is a SOC?
A Security Operations Center (SOC) is responsible for monitoring, detecting, and responding to security threats 24/7.

### Role of a Junior Security Analyst
A Tier 1 SOC Analyst acts as the first line of defense by:
- Monitoring alerts  
- Investigating suspicious activity  
- Escalating incidents when necessary  

✅ Key Idea:  
The main job is **triage**, not solving every incident.

---

## SOC Team Structure
- L1 Analyst → Handles alerts  
- L2 Analyst → Investigates deeper threats  
- SOC Engineer → Maintains tools  
- SOC Manager → Oversees operations  
- Incident Response Team → Handles major incidents  

---

## Alert Workflow
1. Activity is detected  
2. Alert is generated  
3. Analyst investigates  
4. Determine if it's a real threat  
5. Escalate or close  

✅ Key Idea:  
Not all alerts are real and filtering false positives is important.

---

## Takeaways

- SOC analysts focus on monitoring and investigation  
- Cybersecurity is team-based  
- Escalation is a normal part of the workflow  
- Thinking like an analyst is just as important as technical skills
